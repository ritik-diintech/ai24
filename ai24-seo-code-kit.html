01
Essential Meta Tags
Critical — Add First
These are the absolute minimum tags every page on ai24.digital must have. The charset, viewport, title, description, and canonical URL form the foundation of all SEO.
index.html — inside <head>
<!-- ═══════════════════════════════════════════ -->
<!-- AI24 — ESSENTIAL META TAGS                 -->
<!-- ═══════════════════════════════════════════ -->

<!-- Character encoding + viewport (MUST be first) -->
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />

<!-- Page title — appears in Google results & browser tab -->
<title>Ai24 — Enabling Intelligence | AI Solutions for Business</title>

<!-- Meta description — shown in Google results (150–160 chars) -->
<meta
  name="description"
  content="Ai24 enables businesses with AI-powered intelligence, speed, innovation and growth — available 24/7. Formerly Hysus Digital. AI solutions built for the intelligence era."
/>

<!-- Keywords (secondary signal, still useful) -->
<meta
  name="keywords"
  content="AI solutions India, artificial intelligence business, AI company India, Ai24, enabling intelligence, AI automation, business intelligence, Hysus Digital"
/>

<!-- Robots — allow full indexing -->
<meta name="robots" content="index, follow, max-snippet:-1, max-image-preview:large, max-video-preview:-1" />

<!-- Author -->
<meta name="author" content="Ai24 — Hysus Digital Private Limited" />

<!-- Canonical URL — prevents duplicate content issues -->
<link rel="canonical" href="https://www.ai24.digital/" />

<!-- Language -->
<meta http-equiv="content-language" content="en-IN" />

<!-- Theme colour (browser UI accent) -->
<meta name="theme-color" content="#1A56E8" />
📍
Where to paste: Inside the <head> tag of every page. In Next.js, use the <Head> component from next/head. In React, use react-helmet. Update the title and description per page — never use the same description on every page.

Schema.org Structured Data
05
Schema.org — Organisation
Critical for Knowledge Panel
This JSON-LD tells Google exactly who Ai24 is — your name, logo, social links, founding info, and what you do. This is the single most important step toward getting a Google Knowledge Panel for your brand.
index.html — inside <head> or end of <body>
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",

  // ── Core Identity ──────────────────────────
  "name": "Ai24",
  "legalName": "Hysus Digital Private Limited",
  "alternateName": "Ai24 — Enabling Intelligence",
  "description": "Ai24 is an AI intelligence company that enables businesses with four layers of capability: Intelligence, Speed, Innovation, and Growth — available 24/7. Formerly Hysus Digital Private Limited.",
  "slogan": "Enabling Intelligence",

  // ── Web Presence ───────────────────────────
  "url": "https://www.ai24.digital",
  "logo": {
    "@type": "ImageObject",
    "url": "https://www.ai24.digital/logo.png",
    "width": 1200,
    "height": 384
  },
  "image": "https://www.ai24.digital/og-image.png",

  // ── Location ───────────────────────────────
  "address": {
    "@type": "PostalAddress",
    "addressCountry": "IN"
  },

  // ── Contact ────────────────────────────────
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "customer service",
    "email": "hello@ai24.digital",    // ⬛ Update this
    "availableLanguage": ["English", "Hindi"]
  },

  // ── Social Profiles (Google uses these for Knowledge Panel) ──
  "sameAs": [
    "https://www.linkedin.com/company/ai24digital",   // ⬛ Update slug
    "https://www.instagram.com/ai24digital",           // ⬛ Update handle
    "https://twitter.com/ai24digital",                 // ⬛ Update handle
    "https://www.youtube.com/@ai24digital",            // ⬛ Add if exists
    "https://www.facebook.com/ai24digital"              // ⬛ Add if exists
  ],

  // ── What Ai24 Does ─────────────────────────
  "knowsAbout": [
    "Artificial Intelligence",
    "Machine Learning",
    "Business Intelligence",
    "AI Automation",
    "Natural Language Processing"
  ],

  // ── Founding Info ──────────────────────────
  "foundingLocation": {
    "@type": "Place",
    "name": "India"
  }
}
</script>
⚠️
Important: Update every line marked with ⬛ before adding to your site. Wrong social URLs will confuse Google. Test your structured data at search.google.com/test/rich-results after adding.
06
Schema.org — WebSite (Enables Google Sitelinks Search)
Recommended
This enables Google's Sitelinks Searchbox — where users can search your site directly from Google results. Also helps Google understand your website as a distinct entity.
index.html — inside <head> or end of <body>
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Ai24",
  "url": "https://www.ai24.digital",
  "description": "AI intelligence solutions for businesses — Enabling Intelligence.",
  "inLanguage": "en-IN",
  "potentialAction": {
    "@type": "SearchAction",
    "target": {
      "@type": "EntryPoint",
      "urlTemplate": "https://www.ai24.digital/search?q={search_term_string}"
    },
    "query-input": "required name=search_term_string"
  }
}
</script>

08
sitemap.xml
Critical — Missing from Site
A sitemap is the map of all your pages — without it Google doesn't know what to crawl. Update the <loc> entries to match your actual pages. Add new entries every time you add a page.
/public/sitemap.xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
        xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xsi:schemaLocation="http://www.sitemaps.org/schemas/sitemap/0.9
                            http://www.sitemaps.org/schemas/sitemap/0.9/sitemap.xsd">

  <!-- Homepage -->
  <url>
    <loc>https://www.ai24.digital/</loc>
    <lastmod>2025-03-01</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>

  <!-- About page -->
  <url>
    <loc>https://www.ai24.digital/about</loc>
    <lastmod>2025-03-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>

  <!-- Services page -->
  <url>
    <loc>https://www.ai24.digital/services</loc>
    <lastmod>2025-03-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.9</priority>
  </url>

  <!-- Contact page -->
  <url>
    <loc>https://www.ai24.digital/contact</loc>
    <lastmod>2025-03-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.7</priority>
  </url>

  <!-- ⬛ Add one <url> block per page on your site -->

</urlset>
📍
After adding the sitemap: Go to Google Search Console → Sitemaps → Add sitemap and submit https://www.ai24.digital/sitemap.xml. Google will begin crawling your site within 24–72 hours. If using Next.js, use the built-in next-sitemap package instead for automatic generation.
